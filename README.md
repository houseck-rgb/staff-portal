# 최우영씨푸드그룹 앱

파출 정보 및 입사·퇴사 근로자 정보 관리 PWA

## GitHub Pages 배포 방법

1. GitHub에서 새 레포지토리 생성
2. 이 폴더 파일 전체 업로드
3. Settings → Pages → Branch: main / (root) → Save
4. 배포 URL: `https://{username}.github.io/{repo-name}/`

## 폰 홈화면 추가

- **Android**: Chrome에서 열기 → 우상단 메뉴 → "홈 화면에 추가"
- **iPhone**: Safari에서 열기 → 공유 → "홈 화면에 추가"

## 파일 구성

| 파일 | 설명 |
|------|------|
| index.html | 홈화면 (파출정보 / 입사퇴사 버튼) |
| app.html | 메인 앱 |
| manifest.json | PWA 설정 |
| sw.js | 오프라인 캐시 |
| icon-*.png | 아이콘 |
